IMBd Project

In order to gain insights into the performance of movies on IMDb, I developed a Dashboard with Python to analyze IMDb movies by year, genre, movies, total votes, and votes by genre.

Warning : The data presented are partial, as the complete volume was too heavy to load on its own. So we have insights on the movie produced between 2000 and 2005

Overall, the purpose of this dashboard is to analyze the evolution of the movies' performance on IMDb through the years. But also to improve my skills in Data Science and Machine Learning by building a full Python Dashboard.

My next update is to find the best Regression Algorithms to predict the average rating of the soon-coming movies.
