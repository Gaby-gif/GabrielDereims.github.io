Financials Project

On this page, you'll find an interactive Power BI dashboard built using the sample dataset "Power BI Financials." The dashboard is divided into two key sections: an analytical storytelling part that provides a clear, narrative-driven overview of financial performance through dynamic visuals, and a predictive modeling section that leverages regression analysis to forecast future financial trends. This project demonstrates how Power BI can be used not only for descriptive analytics and insight communication, but also as a platform for integrating basic data science techniques such as regression-based prediction.

Data Source : https://learn.microsoft.com/fr-fr/power-bi/create-reports/sample-financial-download
